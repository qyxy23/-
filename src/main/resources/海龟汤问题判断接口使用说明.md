# 海龟汤问题判断接口使用说明

## 🎯 接口概述

海龟汤问题判断接口是基于向量化技术的智能问答系统，能够根据用户问题自动匹配相关线索，并通过AI生成准确的YES/NO/PARTIAL/UNKNOWN判断结果。

## 📋 接口列表

### 1. 完整版接口 (POST)

**接口地址**: `POST /api/haigui/soup/question`

**请求头**:
```
Content-Type: application/json
Authorization: Bearer {your_token}
```

**请求体示例**:
```json
{
  "soupId": "uuid-haigui-soup-id",
  "question": "凶手是男性吗？",
  "topK": 5,
  "includeMatchDetails": true,
  "minSimilarity": 0.3,
  "userId": 12345
}
```

**响应示例**:
```json
{
  "status": "SUCCESS",
  "message": "判断成功",
  "soupId": "uuid-haigui-soup-id",
  "question": "凶手是男性吗？",
  "answer": "YES",
  "explanation": "根据线索显示，凶手确实是男性。 涉及3种类型的线索。",
  "maxSimilarity": 0.85,
  "matchedClueCount": 3,
  "processingTime": 150,
  "matchedClues": [
    {
      "clueId": "clue-123",
      "clueContent": "凶手是一名30岁的男性",
      "clueType": "CHARACTER",
      "similarity": 0.85
    }
  ],
  "vectorMatches": {
    "CLUE": [
      {
        "vectorType": "CLUE",
        "redisKey": "hai_gui:vec:clue:clue-123",
        "contentId": "clue-123",
        "content": "凶手是一名30岁的男性",
        "similarity": 0.85
      }
    ]
  },
  "sessionInfo": {
    "sessionId": "soup-question-1234567890",
    "soupTitle": "密室杀人案",
    "currentProgress": 0.65
  }
}
```

### 2. 简化版接口 (GET)

**接口地址**: `GET /api/haigui/soup/question`

**请求参数**:
```
?soupId=uuid-haigui-soup-id&question=凶手是男性吗？&topK=5&minSimilarity=0.3
```

**响应格式**: 与完整版相同，但不包含详细的匹配信息

## 🔧 参数说明

### 请求参数

| 参数名 | 类型 | 必填 | 默认值 | 说明 |
|--------|------|------|--------|------|
| soupId | String | 是 | - | 海龟汤唯一标识符 |
| question | String | 是 | - | 玩家提出的问题 |
| topK | Integer | 否 | 5 | 返回相关线索数量 (1-20) |
| includeMatchDetails | Boolean | 否 | false | 是否包含详细匹配信息 |
| minSimilarity | Double | 否 | 0.3 | 最小相似度阈值 (0-1) |
| userId | Long | 否 | - | 用户ID，用于统计分析 |

### 响应字段

| 字段名 | 类型 | 说明 |
|--------|------|------|
| status | String | 响应状态: SUCCESS/FAILURE |
| message | String | 响应消息 |
| soupId | String | 海龟汤ID |
| question | String | 原问题 |
| answer | String | AI判断结果: YES/NO/PARTIAL/UNKNOWN |
| explanation | String | AI判断解释说明 |
| maxSimilarity | Double | 最高匹配相似度 |
| matchedClueCount | Integer | 匹配到的线索数量 |
| processingTime | Long | 处理时间(毫秒) |
| matchedClues | Array | 匹配到的线索详情 |
| vectorMatches | Object | 各类型向量匹配结果 |
| sessionInfo | Object | 会话信息 |

### AI判断结果说明

| 结果 | 含义 | 说明 |
|------|------|------|
| YES | 是 | 线索明确支持问题的肯定回答 |
| NO | 否 | 线索明确支持问题的否定回答 |
| PARTIAL | 部分 | 线索部分支持，但不能完全确定 |
| UNKNOWN | 未知 | 现有线索不足以判断 |

## 📚 使用示例

### JavaScript (fetch)

```javascript
// 完整版接口调用
const response = await fetch('/api/haigui/soup/question', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
    'Authorization': 'Bearer ' + token
  },
  body: JSON.stringify({
    soupId: 'your-soup-id',
    question: '凶手是男性吗？',
    topK: 5,
    includeMatchDetails: true,
    minSimilarity: 0.3,
    userId: 12345
  })
});

const result = await response.json();
console.log('AI判断结果:', result.answer);
console.log('解释说明:', result.explanation);
```

### Python (requests)

```python
import requests

# 完整版接口调用
url = 'http://localhost:8080/api/haigui/soup/question'
headers = {
    'Content-Type': 'application/json',
    'Authorization': 'Bearer ' + token
}
data = {
    'soupId': 'your-soup-id',
    'question': '凶手是男性吗？',
    'topK': 5,
    'includeMatchDetails': True,
    'minSimilarity': 0.3,
    'userId': 12345
}

response = requests.post(url, headers=headers, json=data)
result = response.json()

print(f"AI判断结果: {result['answer']}")
print(f"解释说明: {result['explanation']}")
print(f"处理时间: {result['processingTime']}ms")
```

## ⚡ 性能优化建议

### 1. 参数调优

- **topK**: 一般设为3-5，过大会增加响应时间
- **minSimilarity**: 设为0.3-0.5，过滤低质量匹配
- **includeMatchDetails**: 生产环境建议设为false，减少响应大小

### 2. 缓存策略

```javascript
// 相同问题的结果缓存
const cache = new Map();
const cacheKey = `${soupId}_${question}`;

if (cache.has(cacheKey)) {
  return cache.get(cacheKey);
}

const result = await apiCall(soupId, question);
cache.set(cacheKey, result);
```

### 3. 异步处理

```javascript
// 异步调用，避免阻塞UI
async function askQuestion(soupId, question) {
  try {
    const response = await fetch('/api/haigui/soup/question', {
      method: 'POST',
      body: JSON.stringify({ soupId, question })
    });
    return await response.json();
  } catch (error) {
    console.error('问题判断失败:', error);
    return { status: 'FAILURE', message: error.message };
  }
}
```

## 🛡️ 错误处理

### 常见错误响应

```json
{
  "status": "FAILURE",
  "message": "请求参数无效"
}
```

### 错误类型及解决方案

| 错误信息 | 原因 | 解决方案 |
|----------|------|----------|
| 请求参数无效 | 缺少必要参数或参数格式错误 | 检查soupId和question是否提供 |
| 问题向量化失败 | BGE模型处理异常 | 重试或简化问题内容 |
| AI判断生成失败 | AI服务异常 | 检查AI服务状态或使用备用方案 |
| 海龟汤不存在 | soupId无效 | 确认海龟汤ID是否正确 |

### 2. 实时对话集成

```javascript
class SoupGameAI {
  constructor(soupId) {
    this.soupId = soupId;
    this.context = [];
  }

  async ask(question) {
    const response = await apiCall(this.soupId, question);
    this.context.push({ question, answer: response.answer });
    return response;
  }

  getProgress() {
    const yesCount = this.context.filter(c => c.answer === 'YES').length;
    return yesCount / this.context.length;
  }
}
```

### 3. 自定义评分

```javascript
function calculateScore(response) {
  const similarity = response.maxSimilarity;
  const clueCount = response.matchedClueCount;
  const answerType = response.answer;

  // 自定义评分逻辑
  let score = similarity * 0.5;
  score += Math.min(clueCount / 5, 1) * 0.3;
  score += answerType !== 'UNKNOWN' ? 0.2 : 0;

  return score;
}
```
